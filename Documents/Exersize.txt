Задание: "Необходимо протестировать виджет (чат на сайте) на странице https://autofaq.ai.
Тест-кейсы можно оформить в виде чек-листа.

По составленным чек-листам, используя фреймворк Pytest, написать автотесты для Frontend (на Playwright) и Backend. Создавать автотесты по всему чек листу не требуется, достаточно 5-7 тестов.
Для Backend использовать фреймворки не нужно. В ходе написания кода придерживаться паттерна Page Object и принципов ООП. Все тесты должны быть написаны асинхронно (asyncio).

Помимо исходного кода написанные тесты необходимо упаковать в docker контейнер, результатом работы которого будет выполнение тестов и формирование отчета в Allure. Несколько тестов должны намеренно упасть. По упавшим тестам в Allure должно быть видео и скриншот момента падения."
