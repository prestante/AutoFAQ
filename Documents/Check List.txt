Frontend тесты:
	1.	Открытие виджета чата:
	•	Чат открывается при клике на иконку виджета.
	•	Чат корректно отображается на странице (проверка UI: размеры, расположение).
	2.	Отправка сообщения:
	•	Ввод текста и отправка сообщения работает корректно.
	•	Сообщение отображается в чате после отправки.
	3.	Ответ бота:
	•	Бот корректно отвечает на сообщение пользователя.
	•	Ответ отображается в чате с нужным текстом и в правильном формате.
	4.	Закрытие виджета:
	•	Чат можно закрыть по кнопке.
	•	После закрытия чат больше не виден на странице.
	5.	UI валидация:
	•	Текстовые поля, кнопки и элементы интерфейса отображаются корректно.
	•	Ввод недопустимых символов (например, эмодзи, спецсимволы) обрабатывается корректно.

Backend тесты:
	1.	Отправка сообщения через API:
	•	Отправка POST-запроса к API возвращает корректный ответ (200 OK).
	•	Тело ответа содержит правильный ответ бота.
	2.	Обработка некорректного запроса:
	•	Запрос с пустым телом возвращает ошибку 400 Bad Request.
	•	Запрос с некорректными параметрами возвращает корректное сообщение об ошибке.
	3.	Валидация сообщений:
	•	Сервер обрабатывает максимальную длину сообщения.
	•	Сообщения с HTML или JavaScript не проходят.
